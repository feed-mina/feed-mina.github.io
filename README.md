# AI Builder Yerin's Tech Log

민예린의 GitHub Pages 포트폴리오와 Jekyll 기술 블로그입니다.

## Local preview

```bash
bundle install
bundle exec jekyll serve --livereload
```

브라우저에서 `http://127.0.0.1:4000`으로 확인합니다.

## Structure

- `index.html`: 인터랙티브 포트폴리오 메인
- `blog/index.html`: Jekyll 블로그 목록
- `_posts/`: 기술 블로그 글
- `assets/css/styles.css`: 공통 스타일
- `assets/js/site.js`: 프로젝트 카드, 모달, 검색, 테마 전환
