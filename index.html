    비즈니스 로직과
    AI를 연결하는 개발자

    마케팅 데이터 분석에서 시작해 시스템을 설계하고 제어하는 백엔드 엔지니어링으로 확장했습니다. EBS 트래픽 운영, 포스코 DX 메일 보안 운영, B2B 웹 솔루션 개발 경험을 거치며 "사용자가 보는 화면"과 "서버가 책임지는 안정성" 사이의 균형을 배웠습니다.

    최근에는 SDUI 엔진을 통해 화면 구조를 데이터베이스 메타데이터로 추상화하고, Redis 캐싱과 권한별 렌더링으로 배포 없이 UI를 바꾸는 구조를 구현했습니다. AI 도구는 단순 자동화가 아니라, 설계자가 더 넓은 문제를 볼 수 있게 해주는 협업 파트너로 사용합니다.
